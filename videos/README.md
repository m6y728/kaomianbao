# 🎬 视频文件存放目录

## 当前状态
所有卡片当前使用公共 CDN 测试视频。替换为自己的视频：

## 如何替换

### 方法一：上传到 GitHub（推荐）
1. 打开 https://github.com/m6y728/kaomianbao/tree/main/videos
2. 点击 "Add file" → "Upload files"
3. 拖放你的 MP4 文件，命名为 ep01.mp4
4. 提交更改，网站自动生效

### 方法二：使用在线视频 URL
1. 将视频上传到任意视频托管平台
2. 获取直链 URL
3. 修改 index.html 中对应的 src 地址

## 格式要求
- 格式：MP4 (H.264)
- 建议压缩至 10MB 以内以获得更快加载速度
